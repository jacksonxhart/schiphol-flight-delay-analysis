## Project overview

This project analyzes flight operations at Amsterdam Schiphol Airport, with a focus on delays, route patterns, airline activity, aircraft type, terminal assignment, and operational risk. The target audience is airport operations managers and airline station managers interested in identifying where delays are concentrated and which operational factors are most associated with delay risk.

## Files

- `hart_final_project.ipynb`: Final polished Python notebook with data cleaning, EDA, feature engineering, modeling, and conclusions.
- `hart_final_project_sql.ipynb`: Separate SQL notebook containing the required SQL queries and explanations.
- `hart_final_api_pull.ipynb`: Notebook used to pull or prepare the original API data.
- `presentation.pdf`: PDF version of the in-class presentation.

## How to run

Open `hart_final_project.ipynb` in Jupyter Notebook or JupyterLab and run all cells from top to bottom. The SQL notebook can be opened separately to review the required SQL queries.
