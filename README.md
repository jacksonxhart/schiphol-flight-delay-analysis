# schiphol-flight-delay-analysis
Analysis of Schiphol Airport operations (with a focus primarily on delays) for the month ending 28/05/2026.

I'm opting to not upload the .csv files generated by the API pull and the code in the python notebook, because these are date dependent - these can be generated for any date range (within reason) using the code in the API pull file. Additionally, the initial .csv file generated from the API pull is too large to upload to GitHub.
